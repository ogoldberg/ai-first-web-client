Built for AI Agents

Tiered Rendering

Tries the fastest approach first. Content Intelligence (50-200ms), Lightweight JS (200-500ms), full browser only when needed. Automatically picks the right tier.

Procedural Memory

Learns browsing skills from trajectories. Skill versioning with rollback, anti-patterns to avoid mistakes, cross-domain transfer. Gets smarter with every interaction.

API Discovery

Automatically discovers REST and GraphQL endpoints from network traffic. Validates patterns, tracks confidence, enables direct API calls that bypass rendering entirely.

Framework Extraction

Detects Next.js, Nuxt, Gatsby, Remix hydration data. Extracts JSON-LD, OpenGraph, microdata. Gets structured data without parsing HTML.

Session Management

Save and restore authenticated sessions. Health monitoring, automatic refresh, multi-profile support. Sessions persist across restarts.

Stealth & Challenge Detection

Built-in anti-bot evasion with fingerprint rotation, behavioral delays, human-like actions. Detects Cloudflare, PerimeterX, DataDome challenges and attempts auto-bypass.

Confidence Scoring

Every result includes confidence levels. High confidence = direct API calls. Low confidence = full browser. The system learns what works and adapts automatically.

Pagination Detection

Automatically detects pagination patterns (query params, path segments, infinite scroll). Follow pages with a single option, no manual pattern specification needed.

MCP Native

First-class Model Context Protocol support. Use with Claude Desktop, Claude Code, Cursor, Windsurf, or any MCP client. No code generation needed.

Get Started

MCP Clients

Add LLM Browser as an MCP server in any compatible client:

Claude Desktop

Add to ~/Library/Application Support/Claude/claude_desktop_config.json (macOS) or %APPDATA%\Claude\claude_desktop_config.json (Windows):

{
  "mcpServers": {
    "llm-browser": {
      "command": "npx",
      "args": ["llm-browser"]
    }
  }
}

Claude Code (CLI)

Add to ~/.claude/settings.json or project .mcp.json:

{
  "mcpServers": {
    "llm-browser": {
      "command": "npx",
      "args": ["llm-browser"]
    }
  }
}

Cursor

Add to ~/.cursor/mcp.json:

{
  "mcpServers": {
    "llm-browser": {
      "command": "npx",
      "args": ["llm-browser"]
    }
  }
}

Windsurf

Add to ~/.codeium/windsurf/mcp_config.json:

{
  "mcpServers": {
    "llm-browser": {
      "command": "npx",
      "args": ["llm-browser"]

      }
    ]
  }
}

SDK Usage

Use directly in your Node.js application or custom agent:

Node.js SDK

Install and use programmatically:

npm install @llm-browser/core
import { createLLMBrowser } from '@llm-browser/core';

const browser = await createLLMBrowser();
const result = await browser.browse('https://example.com');

console.log(result.content.markdown);
console.log(result.discoveredApis);

await browser.cleanup();

Direct MCP Server

Run as a standalone MCP server:

# Install globally
npm install -g llm-browser

# Run the server
llm-browser

# Or run directly with npx
npx llm-browser

Optional Dependencies

npm install playwright

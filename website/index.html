Web Browsing for AI Agents

A browser where the user is an LLM, not a human. It learns from every interaction, discovers APIs automatically, and progressively optimizes to skip rendering entirely.

Built for AI Agents

Tiered Rendering

Tries the fastest approach first. Content Intelligence (50-200ms), Lightweight (200-500ms), full browser only when needed.

Learns From Usage

Discovers API patterns, learns reliable selectors, builds reusable browsing skills. Gets faster over time.

API Discovery

Automatically discovers REST and GraphQL endpoints from network traffic. Bypass browser rendering with direct API calls.

Session Management

Save and restore authenticated sessions. Health monitoring, automatic refresh, multi-profile support.

Works Without Playwright

Content Intelligence and Lightweight tiers work on any Node.js system. Add Playwright only when you need full browser rendering.

MCP Native

First-class Model Context Protocol support. Use with Claude Desktop or any MCP-compatible client.

How It Works

Get Started

With Claude Desktop

Add to your Claude Desktop config:

{
  "mcpServers": {
    "llm-browser": {
      "command": "npx",
      "args": ["llm-browser"]
    }
  }
}

Programmatic Usage

Use directly in your Node.js application:

import { createLLMBrowser } from '@llm-browser/core';

const browser = await createLLMBrowser();
const result = await browser.browse('https://example.com');

console.log(result.content.markdown);
console.log(result.discoveredApis);

await browser.cleanup();
